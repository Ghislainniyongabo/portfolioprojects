# 📊 Bank Marketing Optimization Project

## 🚀 Overview
This project uses Logistic Regression to predict which customers are most likely to subscribe to a term deposit.

## 🎯 Business Goal
- Reduce marketing costs
- Increase conversion rate
- Target high-value customers

## 📂 Dataset
Bank Marketing dataset (Portuguese banking institution)

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📈 Key Features
- Customer scoring system
- Data visualization
- Business insights
- Exportable priority customer list

## 💡 Results
- Identifies high-probability customers
- Reduces unnecessary calls
- Improves ROI

## 📎 Output
- `priority_customers.csv`
